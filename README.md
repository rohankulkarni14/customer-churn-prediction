# 📊 Customer Churn Prediction

## 📌 Overview
This project focuses on predicting customer churn using machine learning techniques. The goal is to identify customers who are likely to leave a service based on their usage patterns and demographics.

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Scikit-learn  
- Matplotlib  

---

## 📂 Dataset
The dataset contains customer information such as:
- Tenure  
- Monthly Charges  
- Contract Type  
- Internet Services  
- Churn (Target Variable)  

---

## ⚙️ Approach

1. Data Cleaning  
   - Removed unnecessary columns  
   - Handled categorical variables using encoding  

2. Data Preparation  
   - Split data into training and testing sets  

3. Model Building  
   - Applied Logistic Regression  

4. Evaluation  
   - Calculated accuracy  
   - Generated confusion matrix  

---

## 📊 Results

- Achieved an accuracy of ~75–80%  
- Identified key factors influencing churn:
  - Contract type  
  - Monthly charges  
  - Tenure  

---

## 📈 Output

- Feature importance visualization  
- Model accuracy and evaluation metrics  

---

## 🚀 Conclusion
The model demonstrates how machine learning can be used to predict customer churn and help businesses improve retention strategies.
